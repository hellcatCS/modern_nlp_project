# Telegram Manager MVP

MVP userbot для общения с клиентами от лица реального Telegram-аккаунта менеджера.

## Что умеет

- принимает входящие сообщения в личке Telegram
- отвечает от лица реального аккаунта через Telethon
- хранит историю сообщений отдельно по каждому пользователю
- передаёт текст, историю и состояние во внешнюю логику ответа
- может слать алерты в отдельный Telegram-чат
- подходит как обвязка, в которую можно легко подключить другую бизнес-логику

## Текущая архитектура

Проект разделён на две части:

- `main.py` — Telegram-обвязка: получение сообщений, хранение истории, отправка ответов, алерты
- `logic.py` — логика ответа: сюда можно подключать любую свою реализацию
